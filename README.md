### Development

#### Install

Install [`yarn`](https://yarnpkg.com/en/) if you don't already have it available:

```
sudo npm install -g yarn
```

Install the dependencies:

```
yarn
```

#### Build

```
yarn build --watch
```

#### Test

```
yarn test --watch
```

#### Docs

You will need to install jekyll to render and edit the documentation.

```
yarn docs
```
